# AI Chatbot

A personal AI chatbot built with Flask and Groq API.
Runs Llama 70B model. Mobile friendly dark theme UI.

## Features
- Chat with Llama 70B AI
- Mobile responsive design
- Conversation memory
- Clear chat option

## Tech Stack
- Python Flask
- Groq API
- HTML/CSS/JavaScript

## Setup
1. Clone this repo
2. Install requirements: `pip install -r requirements.txt`
3. Add GROQ_API_KEY environment variable
4. Run: `python main.py`
